Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina Linguagem de Script para Web do 2º Período de Análise e Desenvolvimento de Sistemas — IFTM, orientado pelo professor Dr. Ernani Viriato de Melo.

O objetivo deste trabalho é criar um catálogo interativo utilizando HTML, CSS (Tailwind) e JavaScript, reforçando conceitos como manipulação do DOM, filtros, organização semântica e modularização de arquivos.

Funcionalidades do Sistema:

  • Listagem completa de jogos em cards dinâmicos.
  • Sistema de busca textual em tempo real.
  • Filtros por gênero, plataforma e ano.
  • Ordenação por título ou ano de lançamento.
  • Página de detalhes via modal (JavaScript).
  • Dados estruturados em arquivo JSON externo.

Tecnologias Utilizadas:

  • HTML5 com marcação semântica.
  • TailwindCSS (via CDN) para estilização.
  • JavaScript moderno (ES6+) para lógica.
  • JSON para organização dos dados.
